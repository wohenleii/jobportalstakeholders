STUDENT
Register, browse and search jobs, apply with cover letter, bookmark listings and track application status from profile page.
🏢
EMPLOYER
Register company, post job listings with salary and deadline, view applicants and update their application statuses.
🛡️
ADMIN
Full platform control — manage users, approve or close jobs, view live analytics charts and monitor all applications.
⚙️
SYSTEM
JWT auth, bcrypt hashing, role-based middleware, MySQL on Aiven cloud, REST API, job view tracking and pagination.
